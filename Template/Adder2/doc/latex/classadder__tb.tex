\doxysubsection{Detailed Description}
Testbench for a simple 1-\/bit adder \mbox{\hyperlink{classFullAdd}{Full\+Add}}. 

Uses a R\+AM for Input and another R\+AM for expected output. Implements Adder and compares output with Expected Output Ram.
